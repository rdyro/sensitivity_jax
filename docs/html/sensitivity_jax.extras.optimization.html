sensitivity_jax.extras.optimization module¶

sensitivity_jax.extras.optimization.minimize_agd(f_fn, g_fn, *args, verbose=False, verbose_prefix='', max_it=1000, ai=0.1, af=0.01, batched=False, full_output=False, callback_fn=None, use_writer=False, use_tqdm=True)[source]¶

Minimize a loss function f_fn with Accelerated Gradient Descent (AGD) with respect to *args. Uses PyTorch.

Parameters

  • f_fn (Callable) – loss function

  • g_fn (Callable) – gradient of the loss function

  • *args – arguments to be optimized

  • verbose (bool) – whether to print output

  • verbose_prefix (str) – prefix to append to verbose output, e.g. indentation

  • max_it (int) – maximum number of iterates

  • ai (float) – initial gradient step length (exponential schedule)

  • af (float) – final gradient step length (exponential schedule)

  • batched (bool) – whether to optimize a batch of arguments, with batch of losses

  • full_output (bool) – whether to output optimization history

  • callback_fn (Optional[Callable]) – callback function of the form cb_fn(*args, **kw)

  • use_writer (bool) – whether to use tensorflow’s Summary Writer (via PyTorch)

  • use_tqdm (bool) – whether to use tqdm (to estimate total runtime)

Returns

Optimized args or (args, args_hist) if full_output is True

  • batched (bool) – whether to optimize a batch of arguments, with batch of losses

  • ls_pts_nb (int) – number of linesearch points to consider per optimization step

  • force_step (bool) – whether to take any non-zero optimization step even if worse
