sensitivity_jax.sensitivity module¶

sensitivity_jax.sensitivity.generate_optimization_fns(loss_fn, opt_fn, k_fn, normalize_grad=False, optimizations=None, jit=True)[source]¶

Directly generates upper/outer bilevel program derivative functions.

Parameters

  • loss_fn (Callable) – loss_fn(z, *params), upper/outer level loss

  • opt_fn (Callable) – opt_fn(*params) = z, lower/inner argmin function

  • k_fn (Callable) – k_fn(z, *params) = 0, lower/inner implicit function

  • normalize_grad (bool) – whether to normalize the gradient by its norm

  • jit (bool) – whether to apply just-in-time (jit) compilation to the functions

Returns

f_fn(*params), g_fn(*params), h_fn(*params) parameters-only upper/outer level loss, gradient and Hessian.

sensitivity_jax.sensitivity.implicit_hessian(k_fn, z, *params, Dg=None, Hg=None, jvp_vec=None, optimizations=None)[source]¶

Computes the implicit Hessian or chain rule depending on Dg, Hg, jvp_vec.

Parameters

  • k_fn (Callable) – k_fn(z, *params) = 0, lower/inner implicit function

  • z (DeviceArrayBase) – the optimal embedding variable value array

  • *params – the parameters p of the bilevel program

  • Dg (Optional[DeviceArrayBase]) – gradient sensitivity vector (wrt z), for chain rule

  • Hg (Optional[DeviceArrayBase]) – Hessian sensitivity vector (wrt z), for chain rule

  • jvp_vec (Union[DeviceArrayBase, Sequence[DeviceArrayBase], None]) – right sensitivity vector(s) (wrt p) for Hessian-vector-product

  • optimizations (Optional[Mapping]) – optional optimizations

Returns

Hessian/chain rule Hessian as specified by arguments

sensitivity_jax.sensitivity.implicit_jacobian(k_fn, z, *params, Dg=None, jvp_vec=None, matrix_free_inverse=False, full_output=False, optimizations=None)[source]¶

Computes the implicit Jacobian or VJP or JVP depending on Dg, jvp_vec.

Parameters

  • k_fn (Callable) – k_fn(z, *params) = 0, lower/inner implicit function

  • z (DeviceArrayBase) – the optimal embedding variable value array

  • *params – the parameters p of the bilevel program

  • Dg (Optional[DeviceArrayBase]) – left sensitivity vector (wrt z), for a VJP

  • jvp_vec (Union[DeviceArrayBase, Sequence[DeviceArrayBase], None]) – right sensitivity vector(s) (wrt p) for a JVP

  • matrix_free_inverse (bool) – whether to use approximate matrix inversion

  • full_output (bool) – whether to append accumulated optimizations to the output

  • optimizations (Optional[Mapping]) – optional optimizations

Returns

Jacobian/VJP/JVP as specified by arguments
